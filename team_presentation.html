🎯 1. 문제 정의
농업 분야에서 딥러닝 기반 병해충 자동 진단의 필요성

현재의 문제 상황

  • 전 세계 농작물의 약 20~40%가 매년 병해충으로 인해 손실
  • 전문가 육안 진단에 의존 → 지리적·시간적 접근성 한계 존재
  • 비전문가(농민)가 초기 증상을 정확히 구분하기 어려워 대응 시기 지연

⚡
실시간 진단

💡 핵심 가치: 전문 인력 없이도 모바일 디바이스를 통해 누구나 즉시 병해충 종류를 식별하고 조기 대응할 수 있는 기술적 기반을 구축한다.

📈 5-2. 결과 해석
Val Loss < Train Loss 현상의 원인 분석
1단계: 학습(Train) 시 — 의도적 난이도 상향
매 반복마다 입력 이미지에 Rotation, Flip, ColorJitter가 무작위로 적용되어 데이터 일관성이 크게 흐트러짐.
Dropout(p=0.5)이 완전 연결 계층의 50% 뉴런을 무작위 비활성화 → 모델이 가혹한 환경에서 학습하므로 Train Loss는 상대적으로 높게 유지.
2단계: 검증(Eval) 시 — 정제된 평가 환경
데이터 증강이 완전히 배제된 깨끗한 원본 이미지가 입력됨.
model.eval()에 의해 Dropout이 비활성화되어 모든 뉴런의 연결 정보가 온전하게 예측에 참여.
3단계: 결론 — 강인한 일반화 성능 확보
학습 시 가혹한 제약 덕분에 모델이 특정 픽셀에 의존하는 암기형 과적합을 탈피하고,
질병의 핵심 시각적 패턴(반점, 황화 등)을 일반화된 표현으로 내재화함.

✅ 결론: Train Loss > Val Loss 현상은 정규화(Regularization) 기법이 성공적으로 작동한 증거이며, 97.84%의 검증 정확도는 베이스라인 모델로서 매우 우수한 수준의 일반화 성능을 달성했음을 의미합니다.

🚀 6. 향후 계획
모델 고도화 및 설명 가능한 AI(XAI) 적용 로드맵
📌 Phase 1 — 사전학습 모델 비교 실험
ResNet50, EfficientNet-B0, MobileNetV3를 Transfer Learning하여 베이스라인(97.84%) 대비 정확도·파라미터 수·추론 속도를 정량 비교합니다.
📌 Phase 2 — Grad-CAM 시각화 도입
최종 합성곱 계층의 역전파 구배 정보를 활용하여 모델이 잎의 어느 부위(반점, 황화 구역 등)를 주시하여 판단했는지 히트맵으로 시각화합니다.
📌 Phase 3 — 성능 최적화 및 서빙
Learning Rate Scheduler, Early Stopping 등 학습 전략을 정교화하고, 최종 선정 모델을 ONNX 등으로 변환하여 모바일 배포 가능성을 검증합니다.
🔍 Grad-CAM이란?
Gradient-weighted Class Activation Mapping의 약어로, 딥러닝 모델의 블랙박스 문제를 해소하기 위해 예측의 근거가 되는 이미지 영역을 히트맵으로 시각화하는 설명 가능한 AI(XAI) 기법입니다.
🎯 기대 효과
AI 진단 결과의 신뢰도를 객관적으로 검증할 수 있으며, 농민·연구자가 모델의 판단 근거를 직관적으로 이해하여 현장 활용에 대한 신뢰를 확보할 수 있습니다.
